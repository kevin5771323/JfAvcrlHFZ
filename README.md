# 网络安全监控系统 java51

## 项目概述

本项目是基于springboot+mybatis+vue的网络安全监控系统，采用前后端分离的技术架构。支持自定义角色和用户，便于企业内部权限管理。

## 技术栈

### 后端

- Spring Boot
- MyBatis

### 前端

- Vue

## 运行环境

- 开发工具：IDEA
- 数据库：MySQL 5.7

## 部署步骤

1. 确保MySQL 5.7环境已搭建且正常运行。
2. 使用IDEA导入项目，配置数据库连接信息。
3. 运行springboot项目。

## 目录结构

```
├── backend                // 后端代码目录
│   ├── src
│   │   ├── main
│   │   │   ├── java       // 后端Java源代码
│   │   │   ├── resources  // 配置文件目录
│   ├── pom.xml            // Maven项目文件
└── frontend               // 前端代码目录
    ├── src
    │   ├── assets         // 资源文件
    │   ├── components     // Vue组件
    │   ├── views          // 页面视图
    ├── package.json       // Node.js项目文件
```

## 功能模块

- 用户登录/登出
- 自建角色与用户管理
- 网络安全监控

## 系统角色

- 超级管理员：拥有系统最高权限，可管理所有资源。

## 核心亮点

- 前后端分离，便于维护和扩展。
- 权限管理灵活，支持自定义角色与用户。
- 基于成熟的Spring Boot和Vue技术栈，确保系统稳定性和高效性。
